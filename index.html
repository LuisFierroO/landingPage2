CS_01 FINTECH

Conciliación bancaria con IA

  • Problema Equipo financiero: 18 h/semana cuadrando movimientos manualmente, con errores de transcripción.
  • Ejecución Pipeline OCR + LLM revisado por humano, reglas auditables y fallback contable.

CS_02 SALUD

  • Ejecución Modelo de triage explicable + tablero para enfermería con protocolo de override humano.

CS_03 AGRO
